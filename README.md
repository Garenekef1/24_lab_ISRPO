Проект включает практику по темам:

модули import/export
промисы
валидация выражений в калькуляторе

Структура:
index.html
styles.css
js/main.js
js/utils.js
js/math.js
calculator/index.html
calculator/style.css
calculator/main.js
tic-tac-toe/index.html
tic-tac-toe/style.css
tic-tac-toe/main.js
img/.gitkeep

Функциональность:
поддерживаются операции + - * /
выражение собирается из кнопок
ошибки выражения показываются на дисплее
доступны функции работы с массивами и объектами из ES6+ примеров
подключены модули utils.js и math.js
реализован пример промиса для проверки наличия товара

Ограничения:
нет скобок в калькуляторе
нет сложной истории вычислений
tic-tac-toe и calculator в этой работе используются как структура проекта

Архитектура:
index.html и styles.css отвечают за страницу демонстрации
js/main.js содержит практические примеры шагов 6-9
js/utils.js содержит экспорт функций greet add multiply и константу PI
js/math.js содержит экспорт функций square cube и константу E

Логика работы:
деструктуризация извлекает свойства объекта product
spread объединяет массивы и объекты
rest собирает аргументы в функции findMax
модули импортируются через import и используются в main.js
промис checkInventory возвращает успех или ошибку через then/catch
img используется для скриншотов по шагам
